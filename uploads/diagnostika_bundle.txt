ДИАГНОСТИКА СТАРТОВОГО УРОВНЯ
Бланки трёх инструментов курса «Мастерская математического мышления»
Настоящий файл содержит бланки трёх диагностических инструментов курса для проведения входной диагностики в сентябре учебного года.
ДТМ — Диагностический тест математического мышления. Задачный тест на оценку четырёх компонентов: К1 логико-аналитический, К2 комбинаторно-стратегический, К3 пространственно-образный, К4 аргументационный. Три блока по два задания, максимум 15 баллов. Заполняется учеником. Время выполнения — 30–40 минут.
КПН — Карта педагогического наблюдения. Бланк для учителя: фиксация шести параметров поведения каждого ученика на занятии. Шкала 1–3 балла. Заполняется учителем на каждом 2–4-м занятии.
ОПМ — Опросник познавательной мотивации. Анкета из 10 пунктов на две субшкалы: внутренняя мотивация (ВМ) и познавательный интерес (ПИ). Шкала «да / иногда / нет». Для 1–2 классов заполняется устно с цветными карточками; для 3–4 классов — письменно.
Все три инструмента обеспечивают диагностику пяти компонентов математического мышления (К1–К5): К1–К4 — через тест ДТМ и наблюдение КПН; К5 рефлексивный — через КПН и финальное эссе, поскольку рефлексивный компонент имеет процессуальную природу и не диагностируется одноразовым тестом.

3. ОПМ. ОПРОСНИК ПОЗНАВАТЕЛЬНОЙ МОТИВАЦИИ
Опросник состоит из 10 пунктов, разделённых на две субшкалы по 5 пунктов каждая: ВМ (внутренняя мотивация) и ПИ (познавательный интерес). Шкала ответа: «да» — 3 балла, «иногда» — 2 балла, «нет» — 1 балл. Суммарный балл по каждой субшкале: 5–15. Суммарный балл по опроснику: 10–30.
Для 1–2 классов опросник заполняется устно: педагог зачитывает каждое утверждение, ребёнок поднимает цветную карточку (зелёная — «да», жёлтая — «иногда», красная — «нет»). Для 3–4 классов опросник заполняется письменно: ребёнок отмечает выбранный вариант в бланке.
Замечание для учителя. Опросник опирается на двухфакторную модель внутренней мотивации и познавательного интереса (Гордеева, 2021). Формулировки пунктов адаптированы под возраст 7–10 лет и под содержательную специфику курса нестандартных задач.

Использование результатов. Балл ОПМ интерпретируется в связке с баллом ДТМ. Расхождение между уровнями диагностически информативно: высокий ОПМ при низком ДТМ указывает на интерес без сложившихся учебных умений (педагогическая задача — постепенное наращивание сложности); высокий ДТМ при низком ОПМ — на освоенные приёмы без внутренней мотивации (педагогическая задача — поиск содержательного интереса, рекомендация задач на «свою» тему ребёнка).
